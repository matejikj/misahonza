# Task 04: Proces + důvěra + společníci + FAQ + konverzní závěr

## Cíl

Dokončit druhou polovinu homepage tak, aby stránka po wow sekcích převáděla uživatele do důvěry, jistoty a akce.

## Scope

- Proces spolupráce (4-5 kroků)
  - desktop layout se sticky intro
  - mobile stack/accordion
  - volitelně jednoduché zvýraznění aktivního kroku
- Důvěra / hodnoty / čísla / reference
  - 4 hodnotové bloky
  - metriky
  - 1-2 reference
- Sekce O společnících
  - 2 profily + role + quote
- FAQ + péče
  - přístupný accordion
  - copy z draftu
- Finální CTA / kontakt
  - primární + sekundární CTA
  - kontaktní blok a region
  - očekávaná doba odpovědi

## Konverzní požadavky

- CTA musí být konzistentní napříč stránkou
- Finální CTA zve do "prvního kroku" (konzultace / poptávka), ne tvrdý prodej
- Důvěra a proces řeší hlavní bariéry z briefu

## Acceptance Criteria

- Uživatel po průchodu stránkou rozumí průběhu spolupráce a dalšímu kroku
- FAQ pokrývá klíčové otázky (odolnost, koupelna, údržba, čas, cena, výběr)
- Sekce o společnících působí osobně, ne korporátně
- Všechny CTA a kontaktní informace jsou konzistentně stylované a dostupné

## Závislosti

- `task-01` hotový
- `task-02` a `task-03` alespoň v použitelné verzi
