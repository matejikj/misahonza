# Agentní vývojový hub (Sculpted Light)

Tato složka je pracovní základ pro agentní vývoj homepage webu `Sculpted Light`.

## Co je zdroj pravdy

Původní drafty zůstávají v `agent/project-draft/`.
Sjednocený a implementačně použitelný kontext je v `agent/project-context/`.

## Doporučené pořadí čtení pro agenta

1. `agent/project-context/00-start-here.md`
2. `agent/workflow/current-status.md`
3. Aktivní task v `agent/tasks/`
4. `agent/workflow/decision-log.md` (pokud task sahá do scope / architektury)

## Cíl této struktury

- držet kompletní kontext projektu na jednom místě
- rozdělit implementaci do 5 navazujících tasků
- umožnit bezpečný handoff mezi agenty bez ztráty kontextu
