mam tento technicky stack: Astro + Tailwind + (GSAP/Lenis jen v “islands”) Proč je to ideální pro SEO: Astro generuje čisté statické HTML a defaultně posílá minimum JS (skvělé pro LCP/INP). Můžeš mít interaktivitu/animace jen tam, kde je potřeba (např. hero scrolly, configurator, carousel) a zbytek je 0-JS. Skvělé pro “editorial / content-heavy” weby. K tomu: Tailwind (už na to máte DNA) GSAP/ScrollTrigger + Lenis jen pro vybrané sekce (naimportované pouze na stránkách/sekcích, kde běží) Optimalizace obrázků přes @astrojs/image / built-in image pipeline (AVIF/WebP, správné srcset) . Chci mit prezentacni scroll as story hlavni stranku pro dva spolecniky, kteri delaji luxusnu betonove stěrky, dekorativní stěrky a další služby tohoto charakteru
