# Project Brief (Sculpted Light)

## Produkt a typ webu

- Prezentační web / homepage-first experience
- Styl: scroll-as-story, editorial, galerie/interiér vibe
- Nabídka: betonové stěrky, dekorativní stěrky, minerální/dekorativní povrchy a související služby

## Značka a positioning

- Prémiové, klidné, sebevědomé řemeslo
- "Clean luxury" bez tvrdého stavebního vizuálu
- Důraz na světlo, texturu, atmosféru prostoru

## Primární cíle homepage

1. Vyvolat emoci a vizuální touhu
2. Ukázat kvalitu řemesla a rozdíly mezi povrchy
3. Snížit nejistotu (proces, odolnost, údržba)
4. Vybudovat důvěru skrze lidi (2 společníci)
5. Přivést uživatele ke konverzi (konzultace / poptávka / vzorek)

## Primární konverze

- Nezávazná konzultace
- Nezávazná poptávka
- Poslání inspirace / fotek prostoru
- (Volitelně) výběr vzorku / vzorník

## Cílové publikum (pravděpodobné)

- Majitelé bytů/domů se zájmem o kvalitní interiér
- Architekti / interiéroví designéři
- Klienti komerčních interiérů (showroom, studio, kancelář, recepce)

## Klíčové obavy uživatele, které musí homepage vyřešit

- Jaký je rozdíl mezi typy povrchů?
- Hodí se to do mého prostoru (koupelna, obývák, komerce)?
- Je to kvalitní, odolné a jaká je údržba?
- Jak probíhá realizace?
- Komu svěřuji svůj prostor?
- Co mám udělat jako další krok?

## Scope (aktuální)

- Jedna silná homepage se sekcemi 0-10
- Vysoká vizuální kvalita + 2-3 wow motion momenty
- Responsivita desktop/mobile
- Přístupnost a reduced-motion fallbacky
- SEO-friendly obsahová struktura

## Mimo scope (zatím)

- Plný katalog více stránek
- CMS integrace
- Pokročilý konfigurátor povrchů
- Kompletní backend CRM pipeline
